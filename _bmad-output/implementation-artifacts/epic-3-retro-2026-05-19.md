# Retrospective Epic 3 — Live Session Control Room

Tanggal: 2026-05-19
Project: selfstudio
Status Epic: done

## Epic Review

Epic 3 menyelesaikan fondasi live session control room: operator bisa memulai satu session aktif per station, melihat tiga live station cards, mengakhiri session manual/timer, memulihkan state setelah restart, dan melihat detail summary serta local output folder path.

## Story Selesai

- 3.1 Start One Active Session Per Station
- 3.2 Display Three Station Live Cards
- 3.3 End Sessions Manually or by Timer
- 3.4 Recover Live Session State After Restart
- 3.5 Show Session Detail Summary and Local Folder Access

## Yang Berjalan Baik

- Session state sekarang dimiliki backend Go dan persisted secara lokal.
- One-active-session-per-station ditegakkan backend.
- Live cards memakai session list, bukan state frontend buatan.
- SSE invalidation untuk session started/ended sudah menyegarkan dashboard.
- Recovery restart semakin aman dengan persisted active/locked sessions.

## Risiko Dibawa ke Epic 4

- Photo ingestion belum ada; photo count, failure, quarantine, upload status masih placeholder.
- Output path/customer-order sanitization perlu dikeraskan sebelum file routing nyata.
- Timer countdown masih polling/render-based, belum interval halus.
- Manual-end persistence rollback penuh masih deferred sebagai hardening.

## Next Epic Preparation

Epic 4 harus mulai dari watcher/ingestion dengan boundary session lock yang sudah tersedia. Prioritas:

1. Watch station input folders for stable JPGs.
2. Route only to active sessions.
3. Quarantine no-session/late photos.
4. Preserve traceability and duplicate safety.

Keputusan: Epic 3 selesai; lanjut Epic 4 Story 4.1.
